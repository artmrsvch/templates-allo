Поддержка Bluetooth для мобильного управления

Управляйте кофемашиной Nivona CafeRomatica NICR 759 с помощью простого и удобного мобильного приложения NIVONA App. Вы сможете регулировать параметры приготовления кофе, получать полезные советы по эксплуатации, а также контролировать работу машины, используя свой смартфон.

Наслаждайтесь ароматом

Усовершенствованная система сбалансированного аромата Aromatica System позволит получить идеальный по насыщенности напиток, выбирая между тремя ароматическими профилями.

Индивидуальные рецепты

Функция «Мой кофе» позволяет создавать индивидуальные рецепты вашего любимого напитка и сохранять их в памяти машины. Устанавливайте оптимальные для вас настройки и сохраняйте их в своем профиле, чтобы в дальнейшем получать превосходный кофе одним нажатием.
